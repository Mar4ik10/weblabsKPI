Застосування JavaScript в HTML

  1. Схема URL:
    • Вбудований JavaScript у URL для створення закладок з попередньо заповненими полями форми: 
      <a href="javascript:void(prompt('Enter your name:', 'John Doe'));">Click to enter your name</a>
    • Виклик функції JavaScript за допомогою URL:
      • javascript:alert('Hello, World!');
  2. Обробник подій:
    • Обробка події натискання кнопки 
      <button onclick="alert('Button clicked!')">Click me</button>
    • Обробка події наведення миші на елемент: 
      <img src="image.jpg" onmouseover="alert('Mouse over image!')" />
  3. Підстановка (enitty):
    • Виведення динамічної інформації на сторінці за допомогою JavaScript: 
      <p id="dynamicContent"></p>
<script>
    document.getElementById("dynamicContent").innerText = "Current date: " + new Date();
</script>
  4. Вставка (тег SCRIPT):
    • Вбудований JavaScript: 
      <script>
    console.log("Hello, World!");
</script>
    • Підключення зовнішнього JavaScript-файлу: 
      <script src="script.js"></script>
